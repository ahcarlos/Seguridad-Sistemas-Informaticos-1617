def gcd_euclides(a, b)
  while b != 0
    t = b
    b = a % b
    a = t
  end
  a
end

#lehmann_test(n, l) ⇒ Integer

fuente: studentnet.cs.manchester.ac.uk/resources/library/3rd-year-projects/2016/tong.ding.pdf

Test de primalidad de Lehmann Pasos:

  1. Pick a random integer a, 1 <= a < n.

  2. Let c be a**(n−1)/2 mod n.

  3. If c is neither 1 nor n−1, then return n as composite, otherwise store c into an array.

  4. Loop for l times, keep trying different random values of a.

  5. If any element in array is not equal to 1, return n as composite. Otherwise, n is prime.

Parameters:

  • n (Integer) —

    el número que queremos probar si es primo o compuesto

  • l (Integer) —

    el número de veces que se hace la comprobación, ya que es probabilístico

Returns:

  • (Integer) —

    0 si es número primo, 1 si es número compuesto

def lehmann_test(n, l)
  b = []
  for it in 1..l
    a = Random.rand(1..n - 1)
    c = a**((n - 1) / 2) % n
    if c != 1 && c != n - 1
      return 1
    else
      b.insert(it, c)
    end
  end

  return 1 if b.all? { |num| num == 1 } == true
  0
end
